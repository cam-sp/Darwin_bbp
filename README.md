# Darwin_bbp

Files for the submitted article "Assessing the performance of backscattering-based phytoplankton carbon algorithms" by Camila Serra-Pompei, Anna Hickman, Gregory L. Britten and Stephanie Dutckiewicz.

The wrapper to run the figures is "Main_darwin_bbp_2.m"

The file with updates optical parameters to run the Darwin model is  "optics_plankton_new3.txt".

We will create a repository where we can upload the output of the Darwin model as to be able to run all these files (right now the output is too large to be uploaded in GitHub).
